Graphics Illustration

Graphics Illustration created by Wenhan (Cosmos) Zhu and Zixiao Wang. This website is created for holding the graphical illustrations created for CSci 5611 at the University of Minnesota. Graphics Illustration's goal is to host some of our most besting looking animation created for this course and provide a easier way for people who wants to see it.

Details

The illustration created are from 2 platforms. One is a Macbook pro retina 13 2015 and the other is a Dell Inspire laptop. The macbook is powered by an intell core i7 dual core processor with HD 61000 graphics. And the dell laptop is powered with intell core i5 dual core and Nvidia GT 630M. In each illustration, there will be details about which laptop is running the animation and how is the animation working.

Software and Tools

The animations are created by Processing. And the platforms that we developed them are MAC OS X 10.11 and Ubuntu 14.04. This webpage is powered by Materialize.
